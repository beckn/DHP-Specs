### Key features

- Three offer types: SINGLE_EVENT (UC1), BULK_PROCUREMENT (UC0), CAMP (UC2)
- Health service type scoping
- Structured cancellation policy array with per-window refund fractions
- Eligibility criteria list for patient or agency qualification
- BULK_PROCUREMENT conditional requirements: minimumQuantity, maximumQuantity, validityPeriodDays all required
